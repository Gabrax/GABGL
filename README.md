## Getting started
> [!WARNING]
> Dependencies: CMake, C++20 <br>

For general build script info run ```sh build.sh --help```

- Windows <br><br>
Use VS2022 PS/CMD
```bash
sh build.sh -Ccl -Drelease
```
- Linux & MacOS
```bash
sh build.sh -Cclang -Drelease
```
